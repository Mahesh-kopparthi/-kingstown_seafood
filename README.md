# Kings Town Sea Food - Website

A modern, responsive React + Tailwind CSS website for a premium seafood business.

## Features

- **Responsive Design**: Mobile-first design works on all devices
- **Multiple Sections**: Hero, About, Products, Gallery, Why Us, Reviews, Contact
- **Smooth Animations**: Fade-in, floating, and slide animations
- **Contact Form**: Functional contact form with validation
- **Google Maps Integration**: Embedded maps for business location
- **Floating WhatsApp Button**: Quick contact via WhatsApp
- **Product Showcase**: Clean product cards with pricing
- **Customer Reviews**: Testimonials grid with star ratings

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit http://localhost:3000

## Production Build

```bash
npm run build
```

## Customization

Edit the following files to customize:

- **Contact Info**: `Footer.jsx`, `Contact.jsx`
- **Products**: `Products.jsx`
- **Gallery Images**: `Gallery.jsx`
- **Customer Reviews**: `Reviews.jsx`
- **Colors**: `tailwind.config.js`
